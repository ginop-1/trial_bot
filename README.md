### Get it working:
```
git clone https://github.com/ginop-1/trial_bot.git
cd trial_bot
python3 -m venv venv
. venv/bin/activate
pip3 install -r requirements.txt
python3 src/main.py
```

### Usage

Remember to  ``` . env/bin/activate ``` before executing ```main.py``` every time you want to run the program
